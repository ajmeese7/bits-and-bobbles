I fixed this here. The problem was the flex was overriding the default width.

This is what the CSS is now:

.col-md-4 {
  flex: none;
  max-width: 33.3333333333%;
  width: 20%;
}

It successfully sets the width of the columns to 20% instead of 33.3%.
